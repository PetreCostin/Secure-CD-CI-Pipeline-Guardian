# Secure CI/CD Pipeline Guardian

A security-focused application for monitoring and protecting CI/CD pipelines.

## Getting Started

### Prerequisites
- Node.js 16+
- npm 7+

### Installation

```bash
npm install
```

### Build

```bash
npm run build
```

### Run

```bash
npm start
```

### Development

```bash
npm run dev
```

## Features

- Pipeline configuration scanning
- Secret management validation
- Security policy enforcement
- Real-time pipeline monitoring
